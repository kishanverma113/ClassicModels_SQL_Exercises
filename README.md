# 🗃 Database Used

This project uses the **ClassicModels** sample database.

ClassicModels is a relational database that represents a fictional company selling classic model vehicles. It contains realistic business data that is commonly used for SQL practice and interview preparation.

### Main Tables

| Table        | Description                     |
| ------------ | ------------------------------- |
| customers    | Customer information            |
| orders       | Customer orders                 |
| orderdetails | Products included in each order |
| products     | Product details                 |
| productlines | Product categories              |
| employees    | Employee information            |
| offices      | Office locations                |
| payments     | Customer payment records        |

## 🚀 How to Use

### Step 1: Clone the Repository

```bash
git clone https://github.com/kishanverma113/SQL-Practice-Using-ClassicModels.git
```

### Step 2: Open MySQL Workbench

Launch MySQL Workbench and connect to your local MySQL server.

### Step 3: Import the ClassicModels Database

Import the `ClassicModels_Database.sql` file into MySQL.

### Step 4: Execute SQL Files

Run each SQL file in numerical order to understand the concepts step by step.

### Step 5: Practice and Explore

Modify the queries, test different scenarios, and solve additional business problems using the ClassicModels database.
